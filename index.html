Is it already a Vector?

Yes No

It is already a Vector

  • SVGOMG - Online interface for SVGO - minify svg
  • SVGO - NodeJS tool for optimising SVG files
  • Compressor.io - Generic online tool for shrinking images (including SVG)

Not a vector - Trace the rastarized graphic into a Vector

  • PoTrace - Cross platofrm & Open source command line program to convert bitmapped images into vectors
  • Vectorizer.io - Online tool for tracing reastarised images to vector (Good amount of settings e.g. tolerances, bluring and layers)
  • VectorMagic - Adobe Flash or Desktop software for converting rastarised images to vector

Can WebP be used ?

No

PNG if transparent or simple

If the image requires transparancy/opacity, PNG is prefered over GIF (unless IE6 support is needed)

PNG can also produce a smaller file size compared to JPG if the image contains simple information e.g. a logo, or a diagram

PNG JPG

PNG

  • Compressor.io - Generic online tool for shrinking images
  • PngQuant - Command-line utility and library for lossy compression (can be integrated into workflow/build-tool e.g. ImageMin)
  • Photoshop (Save for web) - Try reducing number of colours and bitmap if possible

JPG

  • Lossy (May lose some image quality for much smaller filesize, while the image still being acceptable quality)
    • JPEGMini(Note: Paid for service) - Desktop Software, Adobe Extension & Service/Command-line application
    • MozJpeg - Contains command-line utility for optimising images(cjpeg)(Note: NodeJS ImageMin uses this)
  • Lossless (No image quality should be lost, can normally be done by removing meta info, or improving how the information is stored)
    • JpegTran(Note: Bundled with MozJpeg) - Command-line utility for jpegs includes compression and transforming(e.g. rotating, cropping and scaling)
    • Google - JpegLib - Tool google uses for PageSpeed website
  • Photoshop (Save for web) - Try reducing the quality
